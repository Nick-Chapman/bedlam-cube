# bedlam-cube

Solver for the bedlam cube puzzle.
```text
$ ./bedlam.exe
```

First solution comes in a few seconds...
```text
   eeev   Dvvv   wwzz   fggz
   DeBv   DeCh   Dwgg   ffgz
   ACBB   DCCh   fwCh   fwxh
   AAAB   AyyB   yyxh   yxxx
```
More come after a short wait...
```text
   wwwy   eDwf   DDDf   xDff
   weBy   eehy   xhhf   xxhh
   ACBB   eCCy   vggy   xgzz
   AAAB   ACgB   vCgz   vvvz
```
